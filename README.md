## hassbian系统备份

- 安装homeassistant 配置系统 打包成镜像

## Usage

### 打包镜像
```bash
git clone https://github.com/Ali-IoT-Lab/hassbackup.git
cd hassbackup

#修改权限
 sudo chmod +x rpi-backup.sh
#执行脚本
 sudo sh ./rpi-backup.sh
```
### 备份镜像
```bash
git clone https://github.com/Ali-IoT-Lab/hassbackup.git
cd hassbackup

#修改权限
 sudo chmod +x backup_system.sh
#执行脚本
 sudo sh ./backup_system.sh
```
